## mdbデータ  
### 体長組成面積引延データ2: 曳網面積と海域面積が入っている．漁獲効率が1でない場合があるので注意！！  
### in_漁獲量データ: 生の漁獲量と漁獲尾数のデータ（体長組成は無し）
